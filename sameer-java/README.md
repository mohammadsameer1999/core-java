# Core Java Practice Repository

This repository contains Java programming practice exercises, core Java concepts implementation, and interview preparation materials.

## Repository Structure

### Core Java Concepts (`src/corejava/sameer/`)
- **Abstracts** - Abstract classes and methods implementation
- **Constructor** - Constructor chaining and inheritance examples
- **Exception** - Exception handling examples
- **Lambda Expressions** - Functional programming with lambdas
- **Stream API** - Java 8+ Stream API demonstrations
- **Suppliers** - Supplier functional interface examples
- **Primitive Data Types** - Basic data type examples

### Data Structures & Algorithms (`src/com/sameer/`, `src/com/dsa/`)
- Array operations (reverse, min/max elements)
- Two Sum problem
- Contains Duplicate checker
- Array rearrangement algorithms
- Square array operations

### Interview Preparation (`src/com/interview/`)
- Character count algorithms
- Even number checkers
- Maximum value finder
- Sum calculations
- String case conversions
- Ascending order checks

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/mohammadsameer1999/core-java.git
   ```

2. Navigate to the project directory:
   ```bash
   cd core-java
   ```

3. Compile and run Java files:
   ```bash
   javac src/path/to/YourFile.java
   java YourFile
   ```
